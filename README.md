
# Install Dependencies
`poetry install`
# Start poetry virtual env
`poetry shell`
# Export flask path env
## Linux
`export FLASK_APP=app/webapp.py`

`export FLASK_DEBUG=1`
## Windows
 `set FLASK_APP=app/webapp.py`

`set FLASK_DEBUG=1`
# Create db

- db is already created, just pull from repository

# Run app
`flask run`
